# str2bool3 v1.0.0

## About
Convert string to boolean.
Library recognizes "yes", "true", "y", "t", "1" as True, and "no", "false", "n", "f", "0" as False.
Case insensitive.

## Installation

    $ pip install str2bool3

## Examples
Here's a basic example:

    >>> from str2bool3 import str2bool
    >>> print(str2bool('Yes'))
    True
